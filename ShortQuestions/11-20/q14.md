**14. Which Library is used to create ThreadPool? Which Interface provide main functions of thread-pool?**

The main library used to create ThreadPool is *java.util.concurrent*. This package provides all the high-level concurrency utilities, including ThreadPool management, Executors, Locks, Futures, etc.

The class most commonly used to create thread pools is *java.util.concurrent.Executors*.

The key interface that provides main functions of ThreadPool is *java.util.concurrent.ExecutorService*.
