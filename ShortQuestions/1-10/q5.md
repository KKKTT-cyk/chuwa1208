**5. What is the difference between t.start() and t.run()?**

| Aspect                         | `t.start()`                  | `t.run()`                                  |
| ------------------------------ | ---------------------------- | ------------------------------------------ |
| Creates a new thread?          | ✅ **Yes**                    | ❌ **No**                                   |
| Executes in a separate thread? | ✅                            | ❌                                          |
| Who calls `run()`?             | JVM                          | Current thread                             |
| Call stack                     | New call stack               | Same call stack                            |
| True multithreading            | ✅                            | ❌                                          |
| Typical use                    | **Always** to start a thread | Rarely (mainly for testing or reuse logic) |
